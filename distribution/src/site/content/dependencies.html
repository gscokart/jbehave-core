Dependencies required by JBehave modules

JBehave provides the following artifacts, defined as POM artifacts:

ModuleVersions
jbehave-corelookup
jbehave-groovylookup
jbehave-guicelookup
jbehave-picolookup
jbehave-springlookup
jbehave-antlookup
jbehave-maven-pluginlookup
jbehave-distributionlookup

Once a version has been chosen, each artifact will declare the dependencies required.
